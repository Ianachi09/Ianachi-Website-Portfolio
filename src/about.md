<h3 align="center">First Time Using It</h3>

Starting my first semester on degree, I found a classmates who daily drives Linux. I asked about them more about it and they told me that they first also got lost using it but after a while, their workflow and computer experience got better than on Windows. Around that time, my laptop also started acting up like crashing out of nowhere and my workflow started getting clunky due to my laptop specs. With that, I started researching on what exactly is Linux and how to install it.

After some time (around the end of my first semester), I decided to try out a Linux distro (its like a flavors of Linux ) called Mint Cinnamon. I heard that it is the best distro to start with and beginner-friendly as its UI is similar to Windows, perfect for someone who is recently swapped from it. I tried installing it and surprisingly it was simple, install the ISO image, load it to a USB stick, plug it in and restart your laptop, open BIOS and set required settings, then boot into the ISO which then we can start installing it. For my first time, I decided to try dual-booting Windows and Linux as I had important use case for Windows which is Windows-only software and games. I couldn't be bothered to find alternatives for those so that option is better for now. Dual booting enables me to try out Linux while keep using Windows for daily work. Using it the first time without any experience made me lost. I kept asking to myself, what do I do on Linux as I code fully on Windows. With that, I decided to revert back to Windows only as I don't have any stuff to do on it + dual booting eats my 512GB laptop storage.


Entering my second semester which is more focused on projects, I had to start doing coding more which my laptop couldn't handle with 8GB ram. With that, I had to try using Linux again to make my workflow smoother. I started by learning every commands and basic navigations on Linux and considering what distro to use. Eventually, I decided to stick with Mint Cinnamon as i have prior experience with it. On the time Im installing it, I decided to daily drive Linux fully as I have enough of Windows crashing my laptop. With that, I am fully commited to using and learning Linux. The setup went well (although there was an issue with BIOS part but that was resolved by redoing the steps correctly) and the desktop part was the easy part (due to my experiences on customizing Windows and research). The result of it was worth it, no more laggyness, smoother workflow and personalized desktop. I would choose that over bloated and clunky desktop

<h3 align="center">Expanding my Linux knowledge</h3>

After a few months of using it, I wanted to try another distro for more customabilty and also to learn more. This time, I tried Arch Linux which are more to advanced user. The internet says that this distro is less bloated, but  hard to install and more to DIY (Do it Yourself). With that, I challenged myself to install it. Installing it was hard because there is no DE (Desktop Enviroment) or WM(Window Manager) on install, as it started on terminal so i have to install it manually. I choose Hyprland as WM since it looks good and better for my workflow. After installing Hyprland, I decided to take a break and restart my laptop. Unfortunately, I installed it on the USB drive (live boot instead of my laptop drive) thus I have to stop the installation.

After that situation, my friend decided to told me about CachyOS, an Arch-based distro with configured kernel for gaming and coding. It also comes with installation GUI which makes it easier to install. With that advice, I tried installing CachyOS which is definitely easier that manual installation. It also have tons of preconfigured DE and WM to choose from, which I choose Hyprland again. Completing all the setups, my experiences definitely got better as it is more pesonalized to me.
